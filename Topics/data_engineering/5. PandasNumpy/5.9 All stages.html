Categorizing Pandas Concepts with Plots

Practicing Pandas concepts with plots is an excellent way to solidify your understanding of data manipulation and gain insights from your data. You can categorize your practice into logical steps that mirror a typical data analysis pipeline.

1. Data Loading and Initial Inspection

This is where you bring your data into Pandas and get a first look. Plots here help you understand the basic structure and distribution.

Pandas Concepts:

  • pd.read_csv(), pd.read_excel(), pd.read_sql(), pd.read_json(): Loading various data formats.
  • .head(), .tail(), .sample(): Previewing data.
  • .info(), .shape, .columns, .dtypes: Understanding data types and structure.
  • .describe(): Getting summary statistics for numerical columns.
  • .value_counts(): Counting unique values in categorical columns.

Plotting Concepts (using Matplotlib/Seaborn):

  • Histograms (plt.hist(), sns.histplot()): Visualize the distribution of numerical data (e.g., student marks).
  • Bar Plots (sns.countplot()): Show the frequency of categories (e.g., number of students per grade, per city, or per gender).
  • Scatter Plots (plt.scatter(), sns.scatterplot()): Identify potential relationships between two numerical variables (less common in initial inspection, but good for spotting outliers if you plot index vs. value).

2. Data Cleaning and Preprocessing

Data rarely comes clean. This stage involves handling missing values, duplicates, and incorrect data types. Plots can help identify these issues.

Pandas Concepts:

  • .isnull(), .notnull(), .isna(): Checking for missing values.
  • .dropna(), .fillna(): Handling missing values.
  • .drop_duplicates(): Removing duplicate rows.
  • .astype(): Changing data types.
  • .replace(): Replacing specific values.

Plotting Concepts:

  • Heatmaps (sns.heatmap() with df.isnull()): Visually identify patterns of missing data.
  • Box Plots (sns.boxplot()): Detect outliers in numerical data, which might be data entry errors.
  • Bar Plots: Show the count of unique values before and after cleaning, especially for categorical data.

3. Data Selection and Filtering

This is about focusing on specific subsets of your data for analysis. Plots will then display the characteristics of these subsets.

Pandas Concepts:

  • Column Selection: df['column_name'], df[['col1', 'col2']].
  • Row Selection (Boolean Indexing): df[df['Marks'] > 80], df[(df['Grade'] == 10) & (df['Subject'] == 'Mathematics')].
  • .loc[], .iloc[]: Advanced selection by label or integer location.

Plotting Concepts:

  • After filtering, apply any of the basic plots (histograms, bar plots, scatter plots) to the filtered DataFrame to understand the characteristics of the subset. For example, a histogram of marks only for Grade 12 students.

4. Data Aggregation and Grouping

Summarizing data is a core strength of Pandas. Plots effectively visualize these summaries.

Pandas Concepts:

  • .groupby(): Grouping data by one or more columns.
  • .agg(), .mean(), .sum(), .count(), .min(), .max(), .median(), .std(): Applying aggregation functions.
  • .pivot_table(): Creating summary tables.

Plotting Concepts:

  • Bar Plots (sns.barplot()): Display group-wise statistics (e.g., average marks per subject, total students per city).
  • Line Plots (sns.lineplot()): Show trends over ordered categories (e.g., average marks across grades, if grades were ordered).
  • Box Plots (sns.boxplot()): Compare the distribution of a numerical variable across different categories (e.g., distribution of marks across different grades or subjects).

5. Data Transformation and Feature Engineering

Creating new columns or transforming existing ones can reveal new insights. Plots can show the impact of these transformations.

Pandas Concepts:

  • Creating new columns: df['New_Column'] = df['Col1'] + df['Col2'], df['Pass/Fail'] = np.where(df['Marks'] >= 50, 'Pass', 'Fail').
  • Applying functions: .apply().

Plotting Concepts:

  • Plot the newly created columns (e.g., a bar plot of 'Pass' vs. 'Fail' counts).
  • Use plots to compare distributions before and after transformation (e.g., a histogram of raw marks vs. normalized marks).

6. Advanced Visualization and Insights

Once you're comfortable with the basics, explore more complex visualizations to uncover deeper patterns.

Pandas Concepts:

  • Combining multiple Pandas operations (e.g., filter, group, then transform).

Plotting Concepts:

  • Pair Plots (sns.pairplot()): Visualize relationships between all numerical variables (if you have more).
  • Facet Grids (sns.FacetGrid()): Create multiple plots based on categories (e.g., separate histograms of marks for each gender).
  • Violin Plots (sns.violinplot()): Show distribution and density of data across categories.
  • Stacked Bar Plots: Compare proportions across groups.

sns.histplot(df['Marks'], kde=True, bins=10, color='skyblue')

plt.ylabel('Frequency')

plt.title('Average Marks per Subject')
plt.xlabel('Subject')
plt.ylabel('Average Marks')
plt.xticks(rotation=45, ha='right')
plt.tight_layout()
plt.show()

# 4. Filtering & Plots
grade_12_df = df[df['Grade'] == 12]
avg_marks_grade_12 = grade_12_df.groupby('Subject')['Marks'].mean().sort_values(ascending=False)

sns.barplot(x=avg_marks_grade_12.index, y=avg_marks_grade_12.values, palette='magma')
plt.title('Average Marks per Subject for Grade 12 Students')
plt.xlabel('Subject')
plt.ylabel('Average Marks')
plt.xticks(rotation=45, ha='right')
plt.tight_layout()
plt.show()

# 5. Transformation & Plots
import numpy as np # For np.where

df['Performance_Category'] = np.where(df['Marks'] > 90, 'Excellent',
                                      np.where(df['Marks'] >= 70, 'Good',
                                               np.where(df['Marks'] >= 50, 'Average', 'Needs Improvement')))

plt.figure(figsize=(7, 5))
sns.countplot(x='Performance_Category', data=df, palette='pastel', order=['Excellent', 'Good', 'Average', 'Needs Improvement'])
plt.title('Student Performance Categories')
plt.xlabel('Category')
plt.ylabel('Number of Students')
plt.tight_layout()
plt.show()
