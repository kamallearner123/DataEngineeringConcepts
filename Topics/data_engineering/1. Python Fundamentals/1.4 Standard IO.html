Python Print, Input, and Logging

Print Function: Basic Output

The print() function outputs text or values to the console. It can handle multiple arguments, customize separators, and control line endings.


# Basic print examples
print("Hello, World!")  # Output: Hello, World!
x = 42
print("Value:", x)     # Output: Value: 42
print(1, 2, 3, sep="-") # Output: 1-2-3
print("No newline", end=" ") # Output: No newline (followed by space)
print("Next")         # Output: Next

Input Function: User Input

The input() function prompts the user for input, returning it as a string. An optional prompt can be provided.


# Input examples
name = input("Enter your name: ")  # Prompt: Enter your name
print("Hello,", name)              # Output: Hello, [user input]
age = int(input("Enter your age: ")) # Convert string to integer
print("Age:", age)                 # Output: Age: [user input]

Print with Format Strings

Python supports multiple ways to format strings for printing: f-strings, str.format(), and the % operator. F-strings are the most modern and readable (Python 3.6+).


# F-strings (Python 3.6+)
name = "Alice"
age = 30
print(f"{name} is {age} years old")  # Output: Alice is 30 years old

# str.format()
print("{} is {} years old".format(name, age))  # Output: Alice is 30 years old
print("{0} is {1} years old".format(name, age)) # Output: Alice is 30 years old
print("{n} is {a} years old".format(n=name, a=age)) # Output: Alice is 30 years old

# % operator (older style)
print("%s is %d years old" % (name, age))  # Output: Alice is 30 years old

# Formatting numbers
pi = 3.14159
print(f"Pi to 2 decimals: {pi:.2f}")  # Output: Pi to 2 decimals: 3.14
print("Pi to 2 decimals: {:.2f}".format(pi)) # Output: Pi to 2 decimals: 3.14
print("Pi to 2 decimals: %.2f" % pi)  # Output: Pi to 2 decimals: 3.14

Logging: Structured Output for Debugging

The logging module provides a flexible way to log messages with different severity levels (DEBUG, INFO, WARNING, ERROR, CRITICAL). Itâ€™s more robust than print() for debugging and monitoring.


import logging

# Basic logging setup
logging.basicConfig(level=logging.DEBUG, format='%(asctime)s - %(levelname)s - %(message)s')

# Logging examples
logging.debug("Debug message")    # Output: [timestamp] - DEBUG - Debug message
logging.info("Info message")      # Output: [timestamp] - INFO - Info message
logging.warning("Warning message") # Output: [timestamp] - WARNING - Warning message
logging.error("Error message")    # Output: [timestamp] - ERROR - Error message

# Logging with variables
user = "Bob"
logging.info(f"User {user} logged in")  # Output: [timestamp] - INFO - User Bob logged in

Challenging Questions

Question 1: What will be the output of the following code, and why?


print("A", "B", "C", sep="", end="")
print("D")
Show Answer

Answer: ABCD

Explanation: The first print() uses sep="" to join "A", "B", "C" without spaces, and end="" to prevent a newline. The second print("D") continues on the same line, resulting in "ABCD".

Question 2: What will happen when this code runs, and how can it be fixed to handle invalid input?


age = int(input("Enter your age: "))
print("Age:", age)
Show Answer

Answer: If the user enters a non-integer (e.g., "abc"), the code raises a ValueError: invalid literal for int().

Explanation: The int() function expects a string representing an integer. Non-numeric input causes an error. To fix, use a try-except block to handle invalid input.


try:
    age = int(input("Enter your age: "))
    print("Age:", age)
except ValueError:
    print("Please enter a valid integer")

Question 3: Why does this logging code not display debug messages, and how can it be fixed?


import logging
logging.info("Info message")
logging.debug("Debug message")
Show Answer

Answer: Debug messages are not displayed because the default logging level is WARNING, which is higher than DEBUG.

Explanation: The logging module only outputs messages at or above the configured level. To display DEBUG messages, set the level to DEBUG using logging.basicConfig().


import logging
logging.basicConfig(level=logging.DEBUG)
logging.info("Info message")    # Output: INFO:root:Info message
logging.debug("Debug message")  # Output: DEBUG:root:Debug message
