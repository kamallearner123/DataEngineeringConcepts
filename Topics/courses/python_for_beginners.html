Course Schedule

Session 1: Introduction to Python

Objective: Understand Python basics and set up the development environment.

  • Introduction to Python: History and applications
  • Installing Python on PC or Raspberry Pi; setting up an IDE (e.g., VS Code)
  • Writing and running your first Python program
  • Variables, data types (int, float, string, bool), and basic operations
  • Exercise: Write a program to perform basic arithmetic operations.

Session 2: Control Structures

Objective: Learn to control program flow using conditionals and loops.

  • Conditional statements: if, elif, else
  • Comparison and logical operators
  • Loops: for and while loops
  • Break, continue, and pass statements
  • Exercise: Create a program to check if a number is prime.

Session 3: Functions

Objective: Understand functions and modular programming.

  • Defining and calling functions
  • Parameters and return values
  • Default arguments and keyword arguments
  • Scope and lifetime of variables
  • Exercise: Write a function to calculate the factorial of a number.

Session 4: Lists and Tuples

Objective: Work with Python's list and tuple data structures.

  • Creating and manipulating lists
  • List methods: append, remove, pop, sort, etc.
  • Tuples: Creation, immutability, and use cases
  • List slicing and indexing
  • Exercise: Create a program to manage a to-do list using lists.

Session 5: Dictionaries and Sets

Objective: Explore dictionaries and sets for data organization.

  • Creating and using dictionaries
  • Dictionary methods: keys, values, items
  • Sets: Creation, operations (union, intersection, difference)
  • Practical applications of dictionaries and sets
  • Exercise: Build a program to count word frequency in a sentence.

Session 6: Strings and Web Scraping

Objective: Master string manipulation and learn web scraping basics.

  • String methods: split, join, strip, replace, etc.
  • String formatting: f-strings, .format()
  • Introduction to web scraping and ethics
  • Using requests and BeautifulSoup to scrape data
  • Exercise: Scrape book titles and prices from a sample website.

Session 7: Raspberry Pi GPIO Programming

Objective: Control hardware with Python on Raspberry Pi.

  • Introduction to Raspberry Pi and GPIO pins
  • Setting up RPi.GPIO library
  • Controlling LEDs, buttons, and basic sensors (e.g., temperature sensor)
  • Building simple circuits with Raspberry Pi
  • Exercise: Write a program to control an LED and read a temperature sensor.

Session 8: File Handling and Error Handling

Objective: Learn to handle files and errors in Python.

  • Opening, reading, and writing files
  • Using with statement for file handling
  • Working with CSV files for scraped or sensor data
  • Error handling with try-except for file and web scraping tasks
  • Exercise: Save Raspberry Pi sensor data to a CSV file with error handling.

Session 9: Small Projects

Objective: Apply concepts through hands-on mini-projects.

  • Project 1: Web scraper to collect and analyze news headlines
  • Project 2: Raspberry Pi traffic light simulator using GPIO
  • Project 3: Simple text-based quiz game using dictionaries
  • Integrating lists, dictionaries, and functions
  • Exercise: Complete at least one mini-project and demo it.

Session 10: Capstone Project

Objective: Combine all concepts in a real-world project.

  • Project: Build a Raspberry Pi-based environmental monitor
  • Features: Scrape weather data, read local sensor data, control LEDs, save to file
  • Integrate OOP, file handling, GPIO, and web scraping
  • Present and review projects
  • Exercise: Complete and demo the environmental monitor project.

Designed for students to gain practical Python skills with Raspberry Pi and small projects.
