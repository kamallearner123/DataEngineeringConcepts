Python Lists and Member Functions

Introduction to Lists

Lists in Python are ordered, mutable collections of items, which can hold elements of different data types. They are defined using square brackets [] and support operations like indexing, slicing, and modification. Lists come with built-in member functions (methods) to manipulate their contents.


# Creating and accessing lists
fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # Output: apple
fruits[1] = "blueberry"
print(fruits)  # Output: ['apple', 'blueberry', 'cherry']

List Member Functions

1. append(item)

Adds a single item to the end of the list.


# append example
numbers = [1, 2, 3]
numbers.append(4)
print(numbers)  # Output: [1, 2, 3, 4]
numbers.append("five")
print(numbers)  # Output: [1, 2, 3, 4, 'five']

2. extend(iterable)

Extends the list by appending all elements from an iterable (e.g., list, tuple).


# extend example
list1 = [1, 2]
list2 = [3, 4]
list1.extend(list2)
print(list1)  # Output: [1, 2, 3, 4]
list1.extend((5, 6))
print(list1)  # Output: [1, 2, 3, 4, 5, 6]

3. insert(index, item)

Inserts an item at a specified index, shifting other elements to the right.


# insert example
fruits = ["apple", "cherry"]
fruits.insert(1, "banana")
print(fruits)  # Output: ['apple', 'banana', 'cherry']
fruits.insert(0, "avocado")
print(fruits)  # Output: ['avocado', 'apple', 'banana', 'cherry']

4. remove(item)

Removes the first occurrence of an item from the list. Raises ValueError if the item is not found.


# remove example
colors = ["red", "blue", "red"]
colors.remove("red")
print(colors)  # Output: ['blue', 'red']
# colors.remove("green")  # Raises ValueError: list.remove(x): x not in list

5. pop([index])

Removes and returns the item at the specified index (defaults to the last item if no index is provided).


# pop example
numbers = [10, 20, 30, 40]
last = numbers.pop()
print(last)  # Output: 40
print(numbers)  # Output: [10, 20, 30]
first = numbers.pop(0)
print(first)  # Output: 10
print(numbers)  # Output: [20, 30]

6. clear()

Removes all items from the list, leaving it empty.


# clear example
items = ["pen", "book", "eraser"]
items.clear()
print(items)  # Output: []

7. index(item, [start, [end]])

Returns the index of the first occurrence of an item. Raises ValueError if the item is not found. Optional start and end parameters limit the search range.


# index example
letters = ["a", "b", "a", "c"]
print(letters.index("a"))  # Output: 0
print(letters.index("a", 1))  # Output: 2
# print(letters.index("z"))  # Raises ValueError: 'z' is not in list

8. count(item)

Returns the number of occurrences of an item in the list.


# count example
numbers = [1, 2, 2, 3, 2]
print(numbers.count(2))  # Output: 3
print(numbers.count(4))  # Output: 0

9. sort(key=None, reverse=False)

Sorts the list in place. The key parameter specifies a function for custom sorting, and reverse=True sorts in descending order.


# sort example
numbers = [3, 1, 4, 2]
numbers.sort()
print(numbers)  # Output: [1, 2, 3, 4]
numbers.sort(reverse=True)
print(numbers)  # Output: [4, 3, 2, 1]
words = ["banana", "apple", "cherry"]
words.sort(key=len)
print(words)  # Output: ['apple', 'banana', 'cherry']

10. reverse()

Reverses the order of the list in place.


# reverse example
items = [1, 2, 3]
items.reverse()
print(items)  # Output: [3, 2, 1]

11. copy()

Returns a shallow copy of the list, creating a new list with the same elements.


# copy example
original = [1, 2, 3]
duplicate = original.copy()
duplicate.append(4)
print(original)  # Output: [1, 2, 3]
print(duplicate)  # Output: [1, 2, 3, 4]

Challenging Questions

Question 1: What will be the output of the following code, and why?


lst = [1, 2, 2, 3]
lst.remove(2)
print(lst)
Show Answer

Answer: [1, 2, 3]

Explanation: The remove(2) method removes the first occurrence of 2 from the list. The remaining elements are [1, 2, 3], as the second 2 is not removed.

Question 2: What will be the output of this code, and why? 


lst = ["a", "b", "c"]
lst2 = lst
lst2.append("d")
print(lst)
Show Answer

Answer: ['a', 'b', 'c', 'd']

Explanation: Assigning lst2 = lst creates a reference to the same list object. Modifying lst2 with append("d") also modifies lst because they point to the same memory location.
