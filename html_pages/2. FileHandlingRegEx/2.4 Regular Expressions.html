Topic Subtopics Key Concepts / Examples
Introduction to Regular Expressions What are regular expressions and their purpose Used for pattern matching in text (e.g., validate email, search words)
Basic syntax and patterns Literals (e.g., abc), metacharacters (. ^ $ * + ? { } [ ] \ | ( ))
Using the re module in Python import re, using functions like re.match(), re.search(), etc.
Common RegEx Patterns and Metacharacters Character classes [a-z], [A-Z], \d, \w, \s, .
Quantifiers * (0+), + (1+), ? (0/1), {n}, {n,m}
Anchors ^ (start), $ (end), \b (word boundary)
Special characters and escaping Escape metacharacters with \, e.g., \. for dot, \\ for backslash
Matching and Searching with RegEx re.match() Matches from the start of the string
re.search() Returns the first match anywhere in the string
re.findall() and re.finditer() findall() returns list of all matches, finditer() gives match objects
Case-sensitive vs. case-insensitive matching Use re.IGNORECASE or re.I to make matches case-insensitive
