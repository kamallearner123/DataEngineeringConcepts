Python Function Arguments and Variables

1. Formal Positioning

Formal positioning refers to passing arguments to a function in the order defined in the function's signature. Arguments must match the number and position of parameters.


# Formal positioning
def introduce(name, age):
    print(f"{name} is {age} years old")
introduce("Alice", 30)  # Output: Alice is 30 years old
# introduce(30, "Alice")  # Error: Type mismatch due to wrong order

2. *args (Variable Positional Arguments)

The *args parameter allows a function to accept any number of positional arguments, which are collected into a tuple.


# Using *args
def sum_numbers(*args):
    return sum(args)
print(sum_numbers(1, 2, 3))  # Output: 6
print(sum_numbers(10, 20, 30, 40))  # Output: 100
print(sum_numbers())  # Output: 0

3. Keyword Arguments

Keyword arguments allow passing arguments by specifying parameter names, enabling flexibility in argument order.


# Keyword arguments
def describe_person(name, age):
    print(f"{name} is {age} years old")
describe_person(age=25, name="Bob")  # Output: Bob is 25 years old
describe_person(name="Alice", age=30)  # Output: Alice is 30 years old

4. **kwargs (Variable Keyword Arguments)

The **kwargs parameter allows a function to accept any number of keyword arguments, which are collected into a dictionary.


# Using **kwargs
def print_details(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")
print_details(name="Charlie", age=35, city="Paris")  
# Output: 
# name: Charlie
# age: 35
# city: Paris

5. Global Variables

Global variables are defined outside functions and can be accessed or modified inside functions using the global keyword.


# Global variables
count = 0
def increment():
    global count
    count += 1
    print(f"Count: {count}")
increment()  # Output: Count: 1
increment()  # Output: Count: 2
print(f"Global count: {count}")  # Output: Global count: 2

6. Default Arguments

Default arguments allow parameters to have default values, used when the caller omits those arguments.


# Default arguments
def greet(name, message="Hello"):
    print(f"{message}, {name}!")
greet("Alice")  # Output: Hello, Alice!
greet("Bob", "Hi")  # Output: Hi, Bob!

Challenging Questions

Question 1: What will be the output of the following code, and why?


def print_args(*args, **kwargs):
    print(args)
    print(kwargs)
print_args(1, 2, name="Alice", age=30)
Show Answer

Answer:

(1, 2)
{'name': 'Alice', 'age': 30}

Explanation: *args collects positional arguments (1, 2) into a tuple, and **kwargs collects keyword arguments (name="Alice", age=30) into a dictionary. The function prints both.

Question 2: What will be the output of this code, and why?


x = 10
def modify():
    x = 20
    print(f"Inside function: {x}")
modify()
print(f"Outside function: {x}")
Show Answer

Answer:

Inside function: 20
Outside function: 10

Explanation: Without the global keyword, x = 20 creates a local variable inside the function, leaving the global x unchanged.

Question 3: Why does this code raise an error, and how can it be fixed?


def calc_total(price, tax=0.1, discount):
    return price * (1 + tax) - discount
print(calc_total(100, discount=5))
Show Answer

Answer: The code raises a SyntaxError: non-default argument follows default argument.

Explanation: In Python, non-default arguments (e.g., discount) cannot follow default arguments (e.g., tax=0.1) in a function definition. To fix, place default arguments after non-default ones.


# Fixed code
def calc_total(price, discount, tax=0.1):
    return price * (1 + tax) - discount
print(calc_total(100, 5))  # Output: 104.5
