Python Basic Data Types

Integers (int): Properties and Operations

Integers are whole numbers without a fractional part, with unlimited precision in Python. They support arithmetic and bitwise operations.


# Integer operations
a = 15
b = 4
print(a + b)  # Output: 19
print(a // b) # Output: 3 (floor division)
print(a ** b) # Output: 50625 (exponentiation)

Floating Point (float): Precision and Arithmetic

Floating-point numbers represent real numbers with decimal points. They follow IEEE 754 standards, which can lead to precision issues.


# Float operations
x = 2.5
y = 1.2
print(x * y)  # Output: 3.0
print(x / y)  # Output: 2.0833333333333335
print(0.1 + 0.2)  # Output: 0.30000000000000004 (precision issue)

Complex Numbers (complex): Structure and Operations

Complex numbers consist of a real and an imaginary part, represented as a + bj, where j is the imaginary unit.


# Complex number operations
z1 = 3 + 4j
z2 = 1 - 2j
print(z1 + z2)  # Output: (4+2j)
print(z1 * z2)  # Output: (11-2j)
print(z1.conjugate())  # Output: (3-4j)

NoneType (None): Usage and Properties

None represents the absence of a value or a null value. It is commonly used as a default return value for functions or to indicate uninitialized variables.


# None usage
x = None
print(x)  # Output: None
if x is None:
    print("No value assigned")  # Output: No value assigned

Boolean (bool): Truth Values and Logical Operations

Booleans represent True or False values. They are used in conditional statements and support logical operations like and, or, and not.


# Boolean logical operations
a = True
b = False
print(a and b)  # Output: False
print(a or b)   # Output: True
print(not a)    # Output: False

Callable Data Types

Callable data types are functions that convert or create values of specific types. The following are callable: int(), float(), complex(), and bool(). None is not callable as it is a value, not a function.


# Callable data types
print(int(3.7))      # Output: 3
print(float("5.2"))  # Output: 5.2
print(complex(2, 3)) # Output: (2+3j)
print(bool(0))       # Output: False
# None is not callable
# None()  # Error: 'NoneType' object is not callable

Assigning Variables

Variables in Python are assigned using the = operator. Python is dynamically typed, so types are determined at runtime. Variables can hold any data type and can be reassigned to different types.


# Variable assignment
x = 10          # Integer
y = 3.14        # Float
z = 2 + 3j      # Complex
flag = True     # Boolean
nothing = None  # NoneType
print(x, y, z, flag, nothing)  # Output: 10 3.14 (2+3j) True None
# Reassigning to a different type
x = "Now a string"
print(x)  # Output: Now a string

Operations on Basic Data Types

Basic data types support various operations, including arithmetic (+, -, *, /, //, %, **), comparison (==, !=, >, <, etc.), logical (for booleans), and bitwise (for integers).


# Arithmetic operations
a = 10
b = 3.5
print(a + b)  # Output: 13.5
print(a % b)  # Output: 3.0 (modulus)

# Comparison operations
print(a > b)  # Output: True
print(a == 10)  # Output: True

# Logical operations (Boolean)
x = True
y = False
print(x and not y)  # Output: True

# Complex number arithmetic
z1 = 1 + 2j
z2 = 2 + 3j
print(z1 - z2)  # Output: (-1-1j)

Bitwise Operators (Integers)

Bitwise operators work on integers at the binary level. They include & (AND), | (OR), ^ (XOR), ~ (NOT), << (left shift), and >> (right shift).


# Bitwise operations
a = 10  # Binary: 1010
b = 4   # Binary: 0100
print(a & b)   # Output: 0 (Binary: 0000)
print(a | b)   # Output: 14 (Binary: 1110)
print(a ^ b)   # Output: 14 (Binary: 1110)
print(~a)      # Output: -11 (Binary: Inverts bits, two's complement)
print(a << 2)  # Output: 40 (Binary: 101000)
print(a >> 1)  # Output: 5 (Binary: 0101)

Logical Operators (Booleans and Others)

Logical operators (and, or, not) evaluate expressions to True or False. They can be used with booleans or other types, where non-zero numbers, non-empty sequences, and non-None values are considered True.


# Logical operations
x = True
y = False
print(x and y)      # Output: False
print(x or y)       # Output: True
print(not x)        # Output: False

# Logical operations with non-booleans
a = 5
b = 0
print(a and b)      # Output: 0 (second value if first is truthy)
print(a or b)       # Output: 5 (first truthy value)
print(not a)        # Output: False (a is truthy)

Challenging Questions

Question 1: What will be the output of the following code, and why?


a = 12  # Binary: 1100
b = 5   # Binary: 0101
print(a & b)
Show Answer

Answer: 4

Explanation: The bitwise AND (&) operator compares the binary representations of a (1100) and b (0101). It returns 1 for each position where both bits are 1, resulting in 0100 (decimal 4).

Question 2: What will be the output of this code, and why?


x = 0
y = 10
print(x or y)
Show Answer

Answer: 10

Explanation: The or operator returns the first truthy value or the last value if none are truthy. Here, x is 0 (falsy), so y (10, truthy) is returned.

Question 3: Why does this code produce an error, and how can it be fixed?


x = 3.14
y = 2
print(x & y)
Show Answer

Answer: The code raises a TypeError: unsupported operand type(s) for &: 'float' and 'int'.

Explanation: Bitwise operators like & are only supported for integers, not floats. To fix this, convert the float to an integer if appropriate, or avoid using bitwise operators with floats.


# Example fix: Convert float to int
x = int(3.14)  # Becomes 3
y = 2
print(x & y)   # Output: 2 (Binary: 0011 & 0010 = 0010)
