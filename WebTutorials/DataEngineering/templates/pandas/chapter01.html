Pandas Series and DataFrame Tutorial

Introduction

Pandas is a powerful Python library for data manipulation and analysis. It provides two key data structures:

  • Series: A one-dimensional labeled array for single-column data.
  • DataFrame: A two-dimensional labeled structure for tabular data, like a spreadsheet or SQL table.

This tutorial guides you through creating, manipulating, and analyzing data using Series and DataFrames with practical examples.

Prerequisites

  • Install Python (version 3.6 or higher).
  • Install Pandas: Run pip install pandas.
  • Install NumPy (optional, for numerical operations): Run pip install numpy.
  • Use a Python environment (e.g., Jupyter Notebook, VS Code, or any IDE).

Part 1: Pandas Series

A Series is a 1D labeled array that can hold any data type (integers, strings, floats, etc.). Think of it as a labeled list or a single column.

Step 1: Creating a Series

Create a Series from a list, dictionary, or NumPy array.


import pandas as pd

# Create a Series from a list
data = [10, 20, 30, 40]
series = pd.Series(data, index=['a', 'b', 'c', 'd'])
print(series)

Output:

a    10
b    20
c    30
d    40
dtype: int64

data: The values in the Series.
index: Custom labels (defaults to 0, 1, 2, ... if not specified).

Step 2: Accessing Elements

Access elements using labels or positions.


# Access by label
print(series['b'])  # Output: 20

# Access by position
print(series[1])    # Output: 20

# Slicing
print(series['a':'c'])

Output:

a    10
b    20
c    30
dtype: int64

Step 3: Basic Operations

Series support vectorized operations (element-wise calculations).


# Add 5 to all elements
print(series + 5)

# Filter values greater than 20
print(series[series > 20])

Output:

a    15
b    25
c    35
d    45
dtype: int64

c    30
d    40
dtype: int64

Step 4: Handling Missing Data

Series can handle missing data using NaN.


# Create a Series with missing data
series_with_nan = pd.Series([1, None, 3], index=['x', 'y', 'z'])
print(series_with_nan)

# Check for missing values
print(series_with_nan.isna())

Output:

x    1.0
y    NaN
z    3.0
dtype: float64

x    False
y     True
z    False
dtype: bool

Part 2: Pandas DataFrame

A DataFrame is a 2D labeled data structure with rows and columns, like a table. Each column is a Series.

Step 1: Creating a DataFrame

Create a DataFrame from a dictionary or other data structures.


# Create a DataFrame from a dictionary
data = {
    'Name': ['Alice', 'Bob', 'Charlie'],
    'Age': [25, 30, 35],
    'City': ['New York', 'Los Angeles', 'Chicago']
}
df = pd.DataFrame(data)
print(df)

Output:

       Name  Age         City
0    Alice   25     New York
1      Bob   30  Los Angeles
2  Charlie   35      Chicago

Step 2: Accessing Data

Access columns, rows, or specific values using labels or positions.


# Access a column (returns a Series)
print(df['Name'])

# Access multiple columns
print(df[['Name', 'Age']])

# Access a row by index (using loc)
print(df.loc[1])

Output:

0      Alice
1        Bob
2    Charlie
Name: Name, dtype: object

       Name  Age
0    Alice   25
1      Bob   30
2  Charlie   35

Name          Bob
Age            30
City    Los Angeles
Name: 1, dtype: object

Step 3: Basic Operations

Perform operations like filtering, adding columns, or aggregating.


# Filter rows where Age > 28
print(df[df['Age'] > 28])

# Add a new column
df['Salary'] = [50000, 60000, 75000]
print(df)

# Calculate mean of Age
print(df['Age'].mean())

Output:

       Name  Age         City
1      Bob   30  Los Angeles
2  Charlie   35      Chicago

       Name  Age         City  Salary
0    Alice   25     New York   50000
1      Bob   30  Los Angeles   60000
2  Charlie   35      Chicago   75000

30.0

Step 4: Handling Missing Data

Handle missing data with methods like dropna or fillna.


# Introduce missing data
df.loc[1, 'Age'] = None
print(df)

# Fill missing values with the mean
df['Age'] = df['Age'].fillna(df['Age'].mean())
print(df)

Output:

       Name   Age         City  Salary
0    Alice  25.0     New York   50000
1      Bob   NaN  Los Angeles   60000
2  Charlie  35.0      Chicago   75000

       Name   Age         City  Salary
0    Alice  25.0     New York   50000
1      Bob  30.0  Los Angeles   60000
2  Charlie  35.0      Chicago   75000

Conclusion

This tutorial covered the basics of Pandas Series and DataFrames, including creation, data access, operations, and handling missing data. Series are ideal for single-column data, while DataFrames excel at managing tabular data. Practice these examples in a Python environment to build your data manipulation skills.
